# 算法过程可视化系统

一个基于 Vue 3 的交互式数据结构与算法可视化教学系统。通过动画逐步演示算法的执行过程，并实时高亮对应的 C 语言源代码行，帮助学习者直观理解算法原理。

涵盖 **线性表、栈和队列、树、图、排序、递归与回溯** 六大类，共 **32 个**经典算法与数据结构。

---

## ✨ 功能特性

- **逐步动画演示** —— 把算法每一步的状态变化绘制在 Canvas 上，支持播放 / 暂停 / 单步前进 / 单步回退 / 重置。
- **代码同步高亮** —— 演示过程中实时高亮当前正在执行的代码行，左右对照学习。
- **步骤说明面板** —— 列出全部步骤并附文字说明，可点击任意步骤跳转。
- **多种数据输入** —— 支持手动输入数组 / 图 / 字符串 / 数值，一键随机生成测试数据，并内置预置测试用例。
- **播放速度调节** —— 100ms ~ 2000ms 可调。
- **复杂度信息** —— 每个算法标注时间 / 空间复杂度与难度。
- **键盘快捷键** —— 空格播放、左右方向键单步、R 键重置。
- **大理石奢华主题** —— 暖白大理石背景 + 玄武岩黑 + 金色点缀的高级配色。

```bash

# 3. 构建生产版本（输出到 dist/）
npm run build

# 4. 本地预览构建产物
npm run preview
```

## 📖 使用说明

1. **选择分类** —— 顶部导航栏切换六大算法分类。
2. **选择算法** —— 左侧面板点击具体算法，系统会自动加载默认数据并演示。
3. **输入数据** —— 在「输入数据」区手动输入，或点击「随机生成」/ 预置用例。
   - 数组：英文逗号分隔，如 `38,27,43,3,9`
   - 图：设置节点数与源点，每行一条边 `起点,终点,权重`
   - 字符串：如括号匹配 `{[()]}`、表达式 `3+5*2-1`
   - 数值：如汉诺塔盘子数、棋盘阶数 k
4. **播放控制** —— 使用控制按钮或键盘快捷键观看演示。
5. **对照学习** —— 观察中间画布动画、右侧代码高亮与步骤说明三者联动。

### 键盘快捷键

| 按键 | 功能 |
| --- | --- |
| `空格` | 播放 / 暂停 |
| `→` | 单步前进 |
| `←` | 单步回退 |
| `R` | 重置 |

## 📚 算法清单

### 线性表（7）
顺序表-数组、单链表（带 / 不带头节点）、双链表（带 / 不带头节点）、循环单链表、循环双链表

### 栈和队列（7）
栈-顺序表、栈-链表、队列-顺序表、队列-链表、循环队列、括号匹配、表达式计算

### 树结构（3）
二叉树-链式存储、二叉排序树（BST）、哈夫曼树构造

### 图结构（5）
邻接矩阵、邻接链表、BFS 广度优先搜索、DFS 深度优先搜索、Dijkstra 最短路径

### 排序（6）
冒泡排序、快速排序、直接插入排序、希尔排序、简单选择排序、归并排序

### 递归与回溯（4）
汉诺塔、棋盘覆盖、图着色、迷宫求解

## 📂 目录结构

```
算法可视化/
├── index.html                 # 入口 HTML（含字体引入）
├── vite.config.js             # Vite 配置（端口 5173）
├── package.json
└── src/
    ├── main.js                # 应用入口，挂载 Vue + Pinia
    ├── App.vue                # 主界面：导航栏 + 左侧控制面板 + 画布 + 右侧面板
    ├── style.css              # 全局样式与主题变量（大理石奢华配色）
    ├── algorithms/            # 算法逻辑与步骤生成
    │   ├── registry.js        # 算法注册表（元数据、源代码、分类）
    │   ├── executor.js        # 算法执行调度与随机数据生成
    │   ├── sorting.js         # 排序算法步骤生成
    │   ├── graph.js           # 图算法步骤生成
    │   ├── recursion.js       # 递归与回溯算法步骤生成
    │   └── dataStructures.js  # 线性表 / 栈队列 / 树步骤生成
    ├── store/
    │   └── algorithmStore.js  # Pinia 全局状态（步骤、播放控制、进度）
    └── components/
        ├── VisualizationCanvas.vue  # 核心可视化画布（各类型自适应渲染）
        ├── CodePanel.vue            # 代码面板（当前行高亮）
        └── StepsPanel.vue           # 步骤列表面板
```

> 说明：`components/` 下的 `Header.vue`、`Sidebar.vue`、`Controls.vue`、`MainContent.vue` 为早期版本遗留组件，当前界面未引用。

## 🎨 主题

界面采用「大理石奢华 (Marble Luxury)」配色方案，核心色板定义在 `src/style.css` 的 CSS 变量中：

| 变量 | 色值 | 用途 |
| --- | --- | --- |
| `--bg-dark` | `#F8F6F3` | 大理石暖白背景 |
| `--primary` | `#1A1A1A` | 玄武岩黑（主色 / 当前高亮） |
| `--accent` | `#C9A96E` | 金色点缀 |
| `--text-secondary` | `#8A7968` | 灰褐文字 |
| `--success` | `#85BD73` | 绿色（已访问 / 通过） |
| `--danger` | `#C0564B` | 赤陶红（交换 / 失败） |

如需切换主题，集中修改 `:root` 中的变量即可。
