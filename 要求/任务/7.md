2. 核心概念：图是什么？
2.1 从生活实例理解图
先来看一个例子：

节点：城市（如北京、上海、广州）
边：城市之间的航线
权重：飞行时间（比如北京→上海：2小时）
这种带权重的边，就是我们图算法（如Dijkstra）要处理的核心对象。

2.2 图的数学表示
一个图 G 由两部分组成：

V：节点集合（Vertices）
E：边集合（Edges），每条边连接两个节点，并带有一个权重 w
例如：G = ({A,B,C}, {(A,B,5), (B,C,3), (A,C,2)})

2.3 两种存储结构对比
特性	邻接矩阵	邻接表
存储方式	二维数组 matrix[n][n]	链表数组 adj[n]
空间复杂度	O(n²)	O(n + m) （m为边数）
查询边是否存在	O(1)	O(度)
适合场景	稠密图	稀疏图
实现难度	简单直观	稍复杂
💡 思考题：如果图有100个节点，但只有200条边，你会选择哪种结构？为什么？

class AdjacencyMatrixGraph(Graph):
    def __init__(self, num_nodes, directed=False):
        super().__init__(num_nodes, directed)
        # 初始化一个 num_nodes x num_nodes 的矩阵
        # 初始值设为无穷大（表示无边）
        # 对角线设为0（节点到自身）
        self.matrix = [[float('inf')] * num_nodes for _ in range(num_nodes)]
        for i in range(num_nodes):
            self.matrix[i][i] = 0
请完成以下方法：

add_edge(self, u, v, weight)：在矩阵中设置 matrix[u][v] = weight。如果是有向图，只设置一个方向；如果是无向图，设置两个方向。
get_weight(self, u, v)：直接返回 matrix[u][v]。
get_neighbors(self, u)：遍历第u行，返回所有权重不为无穷大的节点和权重。

class AdjacencyListGraph(Graph):
    def __init__(self, num_nodes, directed=False):
        super().__init__(num_nodes, directed)
        # 每个节点对应一个字典，键是邻居节点，值是权重

请完成以下方法：

add_edge(self, u, v, weight)：在 adj_list[u] 中添加键值对 {v: weight}。无向图则还要在 adj_list[v] 中添加 {u: weight}。
get_weight(self, u, v)：从 adj_list[u] 中获取，如果不存在返回 float('inf')。
get_neighbors(self, u)：返回 adj_list[u].items() 或类似结构。

def manual_input_graph():
    num_nodes = int(input("请输入节点数量："))
    num_edges = int(input("请输入边数："))
    
    # 选择数据结构（这里以邻接表为例）
    graph = AdjacencyListGraph(num_nodes, directed=False)
    
    print("请输入边（格式：u v weight）：")
    for _ in range(num_edges):
        line = input().strip()
        u, v, weight = map(int, line.split())
        graph.add_edge(u, v, weight)
    
    return graph

5. 随机生成图功能
手动输入对于小图还行，但测试算法时我们需要更大的图。让我们实现一个随机生成器。

5.1 设计原则
随机生成的图需要满足：

连通性：任意两个节点之间都有路径（否则Dijkstra算法会出问题）
权重合理：权重在1~100之间（避免0或负数）
边数可控：可以指定边数，或者生成一定密度的图
5.2 实现思路
我们可以分两步走：

先生成一棵生成树：确保图是连通的
再随机添加额外边：增加图的复杂度
python
复制代码
import random

def generate_random_graph(num_nodes, extra_edges=0, directed=False):
    graph = AdjacencyListGraph(num_nodes, directed)
    
    # 第一步：创建生成树（保证连通）
    # 从节点0开始，依次连接下一个节点
    for i in range(1, num_nodes):
        # 随机选择一个已连接的节点作为父节点
        parent = random.randint(0, i-1)
        weight = random.randint(1, 100)
        graph.add_edge(i, parent, weight)
    
    # 第二步：随机添加额外边
    added = set()
    for _ in range(extra_edges):
        u = random.randint(0, num_nodes-1)
        v = random.randint(0, num_nodes-1)
        if u == v or (u, v) in added or (v, u) in added:
            continue  # 避免自环和重复边
        weight = random.randint(1, 100)
        graph.add_edge(u, v, weight)
        added.add((u, v))
    
    return graph
